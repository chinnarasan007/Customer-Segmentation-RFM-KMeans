# Customer-Segmentation-RFM-KMeans
End-to-end customer segmentation project using RFM (Recency, Frequency, Monetary) analysis and K-Means clustering to analyze customer purchasing behavior and identify key customer groups such as loyal, high-value, and potential customers.

# Customer Segmentation using RFM Analysis & K-Means Clustering

## Project Overview

Customer segmentation helps businesses understand different groups of customers based on purchasing behavior.

This project uses **RFM Analysis (Recency, Frequency, Monetary)** combined with **K-Means Clustering** to segment customers into meaningful groups.

---

## Dataset

Online Retail Dataset

The dataset contains transactional data from an online retail store.

Features include:

- InvoiceNo
- StockCode
- Description
- Quantity
- InvoiceDate
- UnitPrice
- CustomerID
- Country

---

## RFM Metrics

Recency  
Days since last purchase.

Frequency  
Number of purchases made by the customer.

Monetary  
Total money spent by the customer.

---

## Clustering Method

We used **K-Means Clustering** to divide customers into segments.

The optimal number of clusters was determined using the **Elbow Method**.

---

## Customer Segments

Example segments identified:

- High Value Customers
- Loyal Customers
- Potential Customers
- At Risk Customers
